1. C'est quoi ce projet ?

Groupie Tracker est une application de bureau (pas web) qui :

  • Récupère des informations sur des artistes musicaux depuis une API
  • Affiche la liste des artistes avec leurs photos
  • Permet de voir les détails de chaque artiste (membres, dates de concerts, etc.)
  • Affiche une carte des lieux de concert

2. Les technologies utilisees

Go (Golang)

  • Langage de programmation créé par Google
  • Compilé, rapide, simple à apprendre
  • Fichier = *.go

Fyne

  • Bibliothèque graphique pour créer des interfaces (fenêtres, boutons, etc.)
  • Alternative à Qt, GTK, etc.
  • Import : fyne.io/fyne/v2

API REST

  • L'application récupère les données depuis : https://groupietrackers.herokuapp.com/api/
  • Format des données : JSON

main.go - Point d'entree

package main

import (
    "groupie-tracker/models"
    "groupie-tracker/ui"
    "fyne.io/fyne/v2/app"
)

func main() {
    myApp := app.New()
    
    myApp.Settings().SetTheme(&ui.CustomTheme{})
    
    win := ui.NewWindow(myApp)
    
    showArtistList(win)
    
    win.Window.ShowAndRun()
}

Fonctions de navigation :

func showArtistList(win *ui.Window) {
    win.ShowLoading("Chargement...")
    
    go func() {
        artists, err := models.FetchArtists()
        if err != nil {
            dialog.ShowError(err, win.Window)
            return
        }
        
        list := ui.NewArtistList(artists, onSelect, onShowMap)
        win.SetContent(list)
    }()
}
Concepts clés :
  • go func() { ... }() = Goroutine (exécution asynchrone)
  • err = gestion des erreurs en Go (toujours vérifier !)

models/artist.go - Les structures de donnees

package models

type Artist struct {
    ID           int      `json:"id"`
    Image        string   `json:"image"`
    Name         string   `json:"name"`
    Members      []string `json:"members"`
    CreationDate int      `json:"creationDate"`
    FirstAlbum   string   `json:"firstAlbum"`
    Locations    string   `json:"locations"`
    
    LocationsList []string `json:"-"`
}

Fetch des données (requête HTTP) :

var (
    cachedArtists []Artist
    lastFetchTime time.Time
    cacheDuration = 5 * time.Minute
)

func FetchArtists() ([]Artist, error) {
    if time.Since(lastFetchTime) < cacheDuration && len(cachedArtists) > 0 {
        return cachedArtists, nil
    }
    
    resp, err := http.Get("https://groupietrackers.herokuapp.com/api/artists")
    if err != nil {
        return nil, fmt.Errorf("erreur: %v", err)
    }
    defer resp.Body.Close()
    
    var artists []Artist
    json.NewDecoder(resp.Body).Decode(&artists)
    
    cachedArtists = artists
    lastFetchTime = time.Now()
    
    return artists, nil
}
Concepts clés :
  • struct = structure (comme une classe sans méthodes)
  • []string = slice (tableau dynamique)
  • map[string][]string = dictionnaire (clé → valeur)
  • defer = exécute à la fin de la fonction
  • json:"name" = tag pour la conversion JSON ↔ Go

ui/window.go - La fenetre principale

package ui

type Window struct {
    Window  fyne.Window
    Content *fyne.Container
}

func NewWindow(app fyne.App) *Window {
    w := app.NewWindow("Groupie Tracker")
    w.Resize(fyne.NewSize(1200, 800))
    w.CenterOnScreen()
    
    content := container.NewMax()
    w.SetContent(content)
    
    return &Window{Window: w, Content: content}
}

func (w *Window) SetContent(content fyne.CanvasObject) {
    fyne.Do(func() {
        w.Content.Objects = []fyne.CanvasObject{content}
        w.Content.Refresh()
    })
}

func (w *Window) ShowLoading(message string) {
    progress := widget.NewProgressBarInfinite()
    label := widget.NewLabel(message)
    
    content := container.NewCenter(
        container.NewVBox(progress, label),
    )
    w.SetContent(content)
}
Concepts clés :
  • fyne.Do() = exécute dans le thread principal (obligatoire pour modifier l'UI)
  • container.NewVBox() = empile verticalement
  • container.NewHBox() = aligne horizontalement
  • container.NewCenter() = centre le contenu

ui/theme.go - Le theme personnalise

package ui

type CustomTheme struct{}

var _ fyne.Theme = (*CustomTheme)(nil)

func (m CustomTheme) Color(name fyne.ThemeColorName, variant fyne.ThemeVariant) color.Color {
    switch name {
    case theme.ColorNameBackground:
        return BgDark
    case theme.ColorNameButton:
        return AccentCyan
    case theme.ColorNameForeground:
        return TextWhite
    default:
        return theme.DefaultTheme().Color(name, variant)
    }
}

func (m CustomTheme) Icon(name fyne.ThemeIconName) fyne.Resource {
    return theme.DefaultTheme().Icon(name)
}

func (m CustomTheme) Font(style fyne.TextStyle) fyne.Resource {
    return theme.DefaultTheme().Font(style)
}

func (m CustomTheme) Size(name fyne.ThemeSizeName) float32 {
    return theme.DefaultTheme().Size(name)
}
Concept : Interface = contrat que doit respecter une structure
